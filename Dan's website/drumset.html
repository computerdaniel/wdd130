Drumset Drumsets Types of Drumsets

There is one way to set up a drum set. You need to first set up "the three" which is the base drum, snare drum and hi hat. After that, then set up the rest of the stuff around the "the three". You can put the rest of the stuff wherever you want. It does take time to set up,to travel with and to take down if you were doing a talent talent show out of town or something like that.
